Countdown widget for Google Sites

Den har GitHub Pages-versionen ar klar att publicera som en statisk sajt. Oppna byggaren, valj datum och design, och kopiera sedan widget-URL:en direkt in i Google Sites.

Oppna byggaren Se exempwidget
1. Publicera Lagg filerna i ett GitHub-repo och aktivera GitHub Pages fran branchen du vill publicera.
2. Bygg Oppna countdown-builder.html. Pa GitHub Pages fylls hostad bas-URL i automatiskt.
3. Badda in Klistra in den genererade widget-URL:en i Google Sites via Infoga > Badda in.
